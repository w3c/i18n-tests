Test passes if the orange rectangle is surrounded by green, and the green margins decrease in the order top, right, bottom, left.

 
  • At the time of writing, there is not yet a consensus that the logical keyword is the approach to be used for this shorthand. This is therefore an exploratory test.
  • A container box has a div inside it. The margin property is applied to that div. It should produce margins on all sides that let through the green colour of the container box.
  • The yellow box has the dimensions 50x50px and sits in a red container that is 150X190px. For the yellow box, padding is set to logical 100px 80px 40px 20px, which should make it fill the container. The yellow box contains another 50x50px square in a lighter colour so that the padding can be seen.
  • The padding should decrease in the order top, right, bottom, left.
