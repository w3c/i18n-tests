  • The Unicode Bidirectional Algorithm causes a Hebrew letter followed by a number in memory to display the number to the left of the letter by default. This test puts a span around the letter, with the dir attribute set, in order to isolate the number from the letter's bidi effect.
  • It repeats this test for a RTL and LTR surrounding context.
  • Reference text is drawn in red behind the test text, and will show through if the order is incorrect. (&#x202d; is LRO, &#x202d; is PDF, א is Hebrew alef.)
