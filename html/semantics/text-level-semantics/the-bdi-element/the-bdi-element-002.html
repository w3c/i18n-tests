‭דדד bbb גגג אאא‬
  • the Unicode Bidirectional Algorithm causes a Hebrew letter followed by a number in memory to display to the left of the letter by default. This test surrounds the number with bdi in order to isolate it from the letter's bidi effect.
  • It repeats this test for a RTL and LTR surrounding context.
  • Reference text is drawn in red behind the test text, and will show through if the order is incorrect. (&#x202d; is LRO, &#x202d; is PDF.)
